# Trabajo Práctico




La tarea consistirá en la creación de un proyecto que sea útil para su desarrollo profesional.

- Promps en cadena que resuelvan algún problema complejo y puedan utilizar como plantilla.
- Creación de un programa que les permita realizar correcciones automáticas.

Deben entregar un documento con el prompt o el código del programa y su enlace a Google Colab (el enlace pueden grabarlo en un fichero .txt).
